## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | hsl(var(--background)) | Background, surfaces |
| Secondary (30%) | hsl(var(--card)) / hsl(var(--muted)) | Cards, sidebar, nav, chat bubbles |
| Accent (10%) | hsl(var(--primary)) | Save buttons, unread indicators |
| Destructive | hsl(var(--destructive)) | Destructive actions only |

Accent reserved for: Primary action buttons (Save Settings, Send Message), active navigation links, and unread message pills.
